# PDFs locais (portfolio)

Apenas ficheiros com **direito de distribuição** devem existir aqui.

- **`codigolimpo.pdf`** — PDF de exemplo **gerado especificamente para este portfólio** (sem conteúdo de terceiros), usado só para demonstrar o fluxo de download protegido por login.
- Outros PDFs não devem ser commitados (ver `.gitignore`: `livros/*.pdf` com exceção para `codigolimpo.pdf`).

Coloque PDFs privados só no seu `.env`/máquina local ou use outro armazenamento; não os envie para repositórios públicos.
